Immurk's Code was a set of pirate traditions and codes of conduct established by the famed pirate Immurk .

History

Immurk's Code was the closest thing to a set of laws, aside from roving gangs of thug law enforcers, used within the city of Immurk's Hold on Dragonisle in the Pirate Isles .
